Wrapperlib
-------------------------------------------------------------------------------

**Wrapperlib** is a library to write different types of wrappers.

The original code and ideas by Tomek Puton (tputon@genesilico.pl).

Next Albert Bogdanowicz did great work to improve the library.

Current Maintainers:
